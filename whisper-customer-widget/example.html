Whisper Chat Widget

智能客服聊天组件使用示例
支持实时消息、文件上传、表情符号等功能

基本使用

主题和位置

代码示例

// 基本初始化 WhisperChat.init({ apiUrl: 'http://localhost:8080/api', websocketUrl: 'ws://localhost:8080/websocket', customerId: 'customer_123', customerName: '张三', theme: 'light', position: 'bottom-right' }); // 打开聊天对话框 WhisperChat.open(); // 关闭聊天对话框 WhisperChat.close(); // 设置主题 WhisperChat.setTheme('dark'); // 设置位置 WhisperChat.setPosition('bottom-left'); // 获取状态 const state = WhisperChat.getState(); console.log(state);

HTML 集成

<!-- 引入 Widget 文件 --> <script src="./dist/whisper-chat-widget.umd.cjs"></script> <link rel="stylesheet" href="./dist/style.css"> <!-- 初始化 Widget --> <script> WhisperChat.init({ apiUrl: 'http://localhost:8080/api', websocketUrl: 'ws://localhost:8080/websocket', customerId: 'your_customer_id', customerName: '客户姓名' }); </script>

配置参数说明

{ apiUrl: 'string', // 必需：API 服务器地址 websocketUrl: 'string', // 必需：WebSocket 服务器地址 customerId: 'string', // 必需：客户唯一标识 customerName: 'string', // 可选：客户姓名 theme: 'light|dark', // 可选：主题，默认 'light' position: 'bottom-right|bottom-left' // 可选：位置，默认 'bottom-right' }
